**How to find dead code:**
```bash
# Search for every file that's required somewhere
grep -r "require(" *.js --include="*.js" | grep -o "'[^']*'" | sort -u

# Compare against the actual file list — anything not required is suspect
ls MA-server/*.js
```

**Key lesson:** Delete before you organize. No point moving dead code
into neat folders.
